# Présentation du projet

Dans le cadre de ma formation au titre professionnel **Développeur Web et Web Mobile (DWWM)**, j’ai réalisé un projet de fin de formation consistant à concevoir et développer une application web.

Le projet, intitulé **Huhu**, est une plateforme web dédiée aux **ventes aux enchères de chevaux en ligne**, accessible aussi bien aux particuliers qu’aux professionnels du secteur équin.

Ce site a pour objectif de proposer une solution numérique moderne permettant de simplifier les ventes et les achats de chevaux via un système d’enchères en ligne.
